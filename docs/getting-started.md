# Getting Started — NVIDIA Resource Suite

**Free. Open. Global. Educational.**

This platform gives every learner on Earth access to the same NVIDIA AI infrastructure
used by researchers, studios, and enterprises — at zero cost.

## Quick Start (5 minutes)

### 1. Clone and install

```bash
git clone https://github.com/influwealth/nvidia-resource-suite
cd nvidia-resource-suite
pip install -r requirements.txt
```

### 2. Set your NVIDIA API Key

Get a free key at [build.nvidia.com](https://build.nvidia.com).

```bash
cp .env.example .env
# Edit .env and add your NVIDIA_API_KEY
```

### 3. Run Hello NIM

```bash
python examples/hello_nim.py
```

You'll see:
- An AI tutor explain the Silk Road to a middle-schooler
- Text embeddings at 4096 dimensions
- A 5-question quiz on Newton's Laws
- Python code generated by CodeLlama-70b

### 4. Explore a World

```bash
python examples/build_world.py
```

This creates a 3D USD scene of East Flatbush Brooklyn, loads the
knowledge base, and runs a RAG query about the history of 458 E 94th St.

### 5. Run the API Server

```bash
python api_server.py
# API docs at http://localhost:7760/docs
```

## No GPU? No problem.

Every module has graceful degradation:

| Module | With GPU | Without GPU |
|--------|----------|-------------|
| NIM LLM | Full inference via NVIDIA cloud | Same (cloud API) |
| NVIDIA Warp | GPU kernels | CPU physics simulation |
| RAPIDS | cuDF/cuML (100x faster) | pandas/sklearn fallback |
| Triton | GPU inference server | Mock inference responses |
| Omniverse | Full USD + RTX render | USD scene files only |
| Earth-2 | Live FourCastNet API | Synthetic weather data |
